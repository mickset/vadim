Политика конфиденциальности

Это демонстрационный документ для тестового проекта Aphrodite Bay. Структура и формулировки подготовлены в соответствии с требованиями 152-ФЗ «О персональных данных» и годятся как шаблон — перед использованием на реальном сайте замените данные оператора и уточните детали у юриста.

1. Общие положения

Настоящая Политика конфиденциальности определяет порядок обработки персональных данных пользователей сайта aphrodite-bay.dezigma.com (далее — «Сайт») оператором персональных данных (далее — «Оператор»).

Используя Сайт и оставляя заявку через форму обратной связи, пользователь подтверждает согласие с условиями настоящей Политики.

2. Какие данные собираются

При заполнении формы «Заказать звонок» Оператор получает: имя, номер телефона. Иные данные (cookie, технические параметры браузера) в текущей версии Сайта не собираются.

3. Цели обработки

Данные используются исключительно для обратной связи с пользователем по вопросам, связанным с объектом недвижимости: консультации, подбор планировки, уточнение стоимости.

4. Правовые основания обработки

Обработка осуществляется на основании согласия субъекта персональных данных, предоставляемого при отправке формы (см. «Согласие на обработку персональных данных»).

5. Хранение и защита данных

Данные хранятся в базе данных Оператора с ограниченным доступом (авторизация, разграничение прав) и не передаются третьим лицам, за исключением случаев, предусмотренных законодательством РФ.

6. Права субъекта персональных данных

Пользователь вправе в любой момент отозвать согласие на обработку персональных данных, запросить сведения об обработке своих данных или потребовать их удаления, направив запрос по контактам, указанным на Сайте.

7. Контакты Оператора

По вопросам обработки персональных данных: sales@aphroditebay.example.
